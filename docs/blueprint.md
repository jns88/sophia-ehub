# **App Name**: Sophia E-Hub

## Core Features:

- Data Ingestion & Calculation Engine: Upload CSV/XLSX files, validate input, and automatically process data using the internal mathematical engine to calculate margin %, net profit, ROAS, strategic score, product status, complaint status, and ABC classification.
- Dynamic Dashboard & Performance Overview: Visualize key performance indicators (KPIs), Top 10 products by margin/loss, an interactive ABC curve, and critical operational alerts for a quick, strategic overview.
- Product Search & Detailed Analysis: Search for products by SKU or name and view a comprehensive detail page presenting all calculated metrics, classifications, and visual status indicators (Approved, Attention, Critical).
- Customizable Report Generator: Create tailored operational and analytical reports by selecting time periods, product scopes (e.g., critical, approved, ABC classes), and specific metrics, with export options to CSV, XLSX, and PDF.
- Integrations Framework: A structured user interface with placeholders for future API integrations with various marketplaces (Mercado Livre, Amazon, Shopee, etc.) and own e-commerce platforms, including fields for connection credentials.
- Personalized Settings & Preferences: Configure company details, upload a logo, select a dark or light theme, customize the main system color, and set default preferences for dashboards and reports to personalize the experience.
- Intelligent Insight Generator: A generative AI tool that analyzes product metrics and statuses (Attention, Critical) to provide concise, actionable text explanations for the underlying issues, aiding in rapid decision-making.

## Style Guidelines:

- Background color: A very dark, subtle blue-purple (#10102D) to establish a sophisticated and analytical environment, suitable for a dark theme and reducing eye strain.
- Primary color: A medium intensity, professional purple-blue (#7070C2) for interactive elements, headlines, and general branding, providing visual distinction against the dark background.
- Accent color: A vibrant, clear cyan-blue (#63DBFF) used judiciously for critical alerts, calls to action, active states, and to highlight key data points on charts, ensuring high visibility.
- Body and headline font: 'Inter' (sans-serif), chosen for its exceptional readability on screens, modern aesthetic, and versatility, supporting clear presentation of data and textual content.
- Clean, crisp, and predominantly monochromatic line icons with occasional accent color usage, to maintain a professional, uncluttered, and highly intuitive user interface.
- A fixed sidebar navigation on the left, an organized main content area utilizing cards for key indicators, clear tabular data displays, and a strong visual hierarchy to ensure an intuitive and pollution-free experience.
- Subtle and purposeful animations for state changes, data loading, and transitions between sections, enhancing perceived performance and user experience without distracting from the core analytical content.
