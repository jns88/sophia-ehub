# 🛒 Sophia E-Hub
### *A colega de trabalho que todo analista merecia ter.*

> Plataforma analítica multicanal para e-commerce — KPIs unificados, Curva ABC e inteligência de dados para Mercado Livre, Amazon, Shopee, Magalu, B2W/Americanas e Cnova.

---

## 📌 Sobre o Projeto

O **Sophia E-Hub** nasceu de uma necessidade real: gestores de e-commerce precisam tomar decisões rápidas com dados espalhados em múltiplos marketplaces, sem tempo para cruzar planilhas manualmente.

A Sophia resolve isso unificando todas as métricas em um único painel inteligente — com diagnóstico automático, alertas de saúde do catálogo e análise de rentabilidade por canal, produto e região.

---

## 🚀 Funcionalidades

### 📊 Dashboard — Gestão à Vista
- KPIs consolidados em tempo real: Tráfego, Conversão, ROI, CAC, LTV, Ticket Médio
- Indicadores de saúde automáticos: `SAUDÁVEL` · `ATENÇÃO` · `CRÍTICO`
- Filtros por período: Hoje, Semana, Mês, Ano

### 🔄 Pipeline de Vendas
- Visualização do funil completo: Tráfego → Conversão → Carrinho → Aprovado → Venda
- Identificação automática de gargalos por etapa

### 📈 Análises Avançadas
- **Curva ABC (Pareto):** segmentação estratégica do catálogo por rentabilidade
- **Performance Mensal:** evolução temporal de KPIs fundamentais
- **Top SKUs por Lucro:** ranking de produtos por margem real
- **Distribuição por Canal:** participação de mercado entre marketplaces

### 🗺️ Performance Regional
- Faturamento, pedidos e ticket médio por estado (UF)
- Classificação ABC por região para priorização de investimento

### 🔔 Saúde do Catálogo e Alertas
- Detecção automática de SKUs com margem negativa (Ruptura Financeira)
- Identificação de oportunidades de escala (Classe A com ROAS acima da média)

### 🔌 Integrações
- Importação via planilha **CSV / XLSX**
- Conexão via **API** com os principais marketplaces brasileiros
- Arquitetura preparada para sincronização multicanal em tempo real

## 💡 Motivação

Este projeto foi desenvolvido com o objetivo de resolver um problema concreto vivido no dia a dia da gestão de e-commerce: a **fragmentação de dados entre canais**.

Analistas e gerentes de e-commerce perdem horas semanais consolidando informações de diferentes plataformas. A Sophia automatiza esse processo, entregando inteligência acionável em segundos.
